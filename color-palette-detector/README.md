# 🎨 Color Palette Extractor

A Python desktop application that extracts dominant colors from images and live webcam input using the K-Means clustering algorithm.

## Features

- Extract dominant colors from uploaded images
- Capture colors from a live webcam
- Adjustable number of color clusters (K)
- Interactive GUI built with PyQt5
- Displays dominant color names and percentage distribution

## Technologies

- Python
- OpenCV
- NumPy
- scikit-learn
- PyQt5
- webcolors

## Installation

Install the required packages:

```bash
pip install -r requirements.txt
```

## Run

```bash
python main.py
```

## Machine Learning Concepts

- Unsupervised Learning
- K-Means Clustering
- Color Quantization
- Image Processing
